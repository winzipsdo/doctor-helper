# Doctor Helpers (医生助手)

一个为临床医生（特别是儿科医生）设计的实用工具集合。使用 React + TypeScript 开发，支持离线访问的 PWA 应用。

## 🌟 特性

- 💻 响应式设计，支持各种设备
- 📱 支持 PWA，可离线使用
- 🔒 纯前端计算，无需后端，保护患者隐私
- 🎯 专注临床实用工具
- 🔍 直观的用户界面

## 📱 当前工具

- 新生儿 Apgar 评分计算器
  - 支持五项指标评估：心率、呼吸、肌张力、反应力和肤色
  - 实时计算总分
  - 自动评估建议

## 🛠️ 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- PWA (支持离线访问)
